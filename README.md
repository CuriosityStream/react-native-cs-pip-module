# react-native-cs-pip-module

PictureInPicture support

## Installation

```sh
npm install react-native-cs-pip-module
```

## Usage

```js
import CsPipModule from "react-native-cs-pip-module";

// ...

const result = await CsPipModule.multiply(3, 7);
```
